# 大富翁桌游助手

配合实体地图的多人桌游助手，完全离线使用！

## 功能特点

- 📱 移动端优化，直接在手机浏览器打开
- 🚫 完全离线，无需联网
- 👥 支持2-4人游戏
- 🎲 自带骰子系统
- 🏠 完整的买地、建房、收租系统
- 🎭 随机事件系统
- 💾 自动保存游戏进度

## 如何使用

### 1. 直接打开

用手机浏览器打开 `index.html` 文件即可使用

### 2. 添加到主屏幕 (推荐)

在手机浏览器中选择"添加到主屏幕"，可以像App一样使用

## 游戏玩法

1. 选择玩家数量和名称，开始游戏
2. 当前玩家点击"摇骰子"
3. 根据骰子点数在实体地图上移动棋子
4. 在App中选择对应格子编号
5. 进行买地、建房等操作
6. 点击"结束回合"交给下一位玩家

## 地图配置

包含32格标准地图：
- 22个城市地产
- 3个交通设施
- 2个公共设施
- 机会/公共事件格
- 监狱系统
- 起点/免费停车

## 技术栈

- 纯 HTML/CSS/JavaScript
- LocalStorage 本地存储
- PWA 支持

## 文件说明

- `index.html` - 主页面
- `styles.css` - 样式文件
- `game.js` - 游戏逻辑
- `manifest.json` - PWA配置
